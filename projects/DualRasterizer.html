Dual Rasterizer


Goal

Writing an application that can switch between CPU and GPU based rasterization and achieve the same lighting on a provided model. For the CPU, the several rasterization stages have to be written by hand in addition to the shaders themselves. The GPU side has to be set up and the effects have to be written in HLSL.
